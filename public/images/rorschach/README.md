# Planches Rorschach (quiz Assistant AutiCare)

Pour utiliser les **planches officielles** du test de Rorschach (photos) dans le quiz, placez ici des images nommées :

- `plate_1.jpg` (ou .png, .svg, .webp) — planche 1
- `plate_2.jpg` — planche 2
- `plate_3.jpg` — planche 3
- `plate_4.jpg` — planche 4

Le quiz affiche 4 images ; vous pouvez donc utiliser les planches 1 à 4 (ou 1 à 10 si vous souhaitez étendre plus tard).

## Où télécharger les planches (domaine public)

Les planches originales de Hermann Rorschach (1921) sont dans le **domaine public**. Vous pouvez les télécharger depuis :

- **Wikimedia Commons** : [Rorschach test](https://commons.wikimedia.org/wiki/Rorschachtest)  
  Fichiers individuels : `Rorschach_blot_01.jpg` … `Rorschach_blot_10.jpg`

Après téléchargement, renommez-les en `plate_1.jpg`, `plate_2.jpg`, `plate_3.jpg`, `plate_4.jpg` et placez-les dans ce dossier.

Si aucun fichier n’est présent, l’application génère automatiquement des taches d’encre en SVG (noir et blanc) en secours.
