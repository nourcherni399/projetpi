# AutiCare – Autism Support Web Platform

## Overview
This project was developed as part of a university academic program in Software Engineering (Academic Year 2025–2026).

AutiCare is a full-stack web application designed to support individuals with autism by providing a safe and accessible digital environment. The platform allows users to order specialized products, read awareness modules, write articles, register for thematic events, and book appointments with autism specialists.

## Features
- Product ordering system for autism-related products
- Awareness and educational modules
- Article writing and publishing
- Thematic event registration
- Appointment booking with specialists
- Secure authentication and role management (Admin, User, Specialist)

## Tech Stack

### Frontend
- Twig
- HTML5
- CSS3
- JavaScript

### Backend
- PHP
- Symfony Framework
- Doctrine ORM
- MySQL

## Architecture
The application follows the MVC (Model-View-Controller) architecture provided by Symfony:
- Model: Manages database entities (Users, Products, Articles, Events, Appointments, Modules)
- View: Handles the user interface using Twig templates
- Controller: Manages business logic and user requests

## Getting Started

### Prerequisites
- PHP 8+
- Composer
- Symfony CLI
- MySQL

### Installation
1. Clone the repository
2. Run `composer install`
3. Configure the `.env` file
4. Create the database
5. Run migrations
6. Start the Symfony server
